# 布洛芬 (Ibuprofen) 药品资料 (示例/合成数据)

> 本文件为学习演示用的合成资料，非真实医学建议。临床用药请遵医嘱。

## 适应症
布洛芬是非甾体抗炎药 (NSAID)，用于缓解轻中度疼痛 (头痛、牙痛、痛经、
肌肉骨骼疼痛) 及退热、抗炎。

## 常用剂量
- 成人镇痛：每次 200～400 mg，每 4～6 小时一次，必要时服用。
- 每日最大剂量一般不超过 1200 mg (非处方) 或 3200 mg (处方监护下)。
- 建议餐后服用以减少胃肠刺激。

## 禁忌与注意
- 活动性消化性溃疡、严重心衰、严重肾功能不全者慎用或禁用。
- 长期大剂量增加心血管事件与消化道出血风险。
- 妊娠晚期禁用。

## 药物相互作用
与抗凝药 (如华法林) 联用增加出血风险；与其他 NSAID 或糖皮质激素联用增加胃肠损伤。
与某些降压药联用可能削弱降压效果。
