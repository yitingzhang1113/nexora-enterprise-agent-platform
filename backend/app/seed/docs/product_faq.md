# Nexora 平台产品 FAQ（示例/合成数据）

## Nexora 是什么
Nexora 是一个企业级 AI Agent 平台，提供基于知识库的检索增强问答（RAG）、
业务工具调用（MCP）、多轮对话记忆与可观测性，帮助企业快速搭建智能助手。

## 支持哪些数据源
当前支持上传 PDF / TXT / Markdown 文档，以及抓取网页。后续可扩展更多连接器
（如网盘、知识库、工单系统）。

## 计费方式
- 免费版：单工作区，最多 1000 个文档分片，社区支持。
- 团队版：多工作区、SSO、优先支持，按席位月付。
- 企业版：私有化部署、SLA、专属客户成功经理。

## 数据安全
所有文档默认仅在你的工作区内可见；企业版支持私有化部署，数据不出内网。

## 如何接入自己的大模型
平台通过统一的模型路由层支持多家提供商（本地 Ollama、Claude、OpenAI 等），
在管理后台配置模型与密钥即可切换。
